# RAG Pipeline Examples

Code examples for the blog post: [Building a Multi-Agent RAG Pipeline](https://nivantlabs.com/blog/multi-agent-rag-pipeline)

## Files
- `query_router.py` — Query classification router
- `hybrid_retriever.py` — Dense + sparse hybrid retrieval
- `reranker.py` — Cross-encoder reranking

## Usage
```bash
python3 -m venv venv
source venv/bin/activate
pip install numpy
```
